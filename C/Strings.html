strings

A group of characters defined between double quotation marks is a string. It is a constant string ,But in C language, a string variable is nothing but an array of characters and terminated by a null character (\0).

Declaration

char identifier [size] ;
Eg : char st[20] ;

Initialization

At the time of declaring a string variable we can store a constant string into that variable is called as initialization of a string.

syntax

char identifier[size] = "string";

Eg : char st[10] = "WELCOME";
char str[ ] = "ABCD" ;

note

The compiler assigns a constant string to a string variable, it automatically supplies a null character (\0)at the end of a string. Therefore the size should be equal to number of characters in the string plus 1. In C language, string initialization is possible, but string assigning is not possible.

code

#include<stdio.h>
#include<conio.h>
void main()
{
  char st[10]="WELCOME";
  char str[]="ABCD";
  printf("st = %s",st);
  printf("\nstr = %s",str);
}

output

st = WELCOME
str = ABCD

gets

It gets a string from stdin.

syntax

char *  gets(char  *s);

Eg :
1) gets(st);
2) gets(“welcome”); -> not possible

puts

It outputs a string to stdout(Monitor) and appends a new line character.

syntax

char *  puts(const char *s);

Eg :
1) puts(st);
2) puts("welcome"); -> possible

code

#include<stdio.h>
#include<conio.h>
void main()
{   
  char st[20];
  puts("Enter a String : ");
  gets(st);
  puts(st);
}

output

Enter a String : welcome
welcome

getch

It gets a character from the console but doesnot echo (print) to the screen

syntax

int  getch();

getche

It gets a character from the console and echoes to the screen.

syntax

int  getche();

code

#include<stdio.h>
#include<conio.h>
void main()
{   
  char ch,che;  
  printf("Enter first character  : ");
  ch=getch();
  printf("Enter second character : ");
  che=getche();
  printf("First character  : %c \n"ch);
  printf("Second character : %c"che);
}

output

Enter first character  : r
Enter second character  : p
First character  : r
Second character : p

note

No need to press enter key after entering a character.

Two dimensional character arrays (String array)

A list of strings can be treated as a table of strings and a two dimensional character array can be used to store the entire list.
For example :
char st[20][20] may be used to store a list of 20 strings, each of length not more than 20 characters.

code

#include<stdio.h>
#include<conio.h>
void main()
{
  char st[20][20];
  int n,i;
  printf("Enter number of strings : ");
  scanf("%d",&n);
  printf("Enter %d strings : ",n);
  fflush(stdin);
  for(i=0;i< n;i++)
  {
    gets(st[i]);
  }
  printf("\nGiven strings : ");
  for(i=0;i< n;i++)
  {
    puts(st[i]);
  }
}

output

Enter number of strings : 2
Enter 2 strings :
welcome
codetocook
Given strings :
welcome
codetocook
