constants

Constants in C refer to fixed values that donot change during the execution of a program. "const" is a keyword and is used to declare constants.

syntax

const datatype identifier=value;
//(or)
const datatype identifier-1=value,….,identifier-n=value;

Eg:

const int a=100;
const int x=10,y=20,z=30;

code

#include<stdio.h>
#include<conio.h>
void main()
{
  const int a=100;
  printf("a=%d",a);
   // a=200
  printf("\na=%d",a);
}

output

a=100
a=100

note

In the above program we give a=200, we will get an error can’t modify a constant object

Symbolic constants:

#define is a pre-processor statement and is used to define symbolic constants.

syntax

#define identifier value

Eg:

#define pi 3.14
#define  g   9.8

code

#include<stdio.h>
#include<conio.h>
#define pi 3.14
#define g 9.8
#define val 100
void main()
{
  printf("pi = %.2f ",pi);
  printf("g = %.1f ",g);
  printf("val = %d",val);
}

output

pi=3.14
g=9.8
val=100
