static data member

A data member of a class can be declared as static, the properties of a static member variables are similar to a C static variable. It is initialized to zero, when the first object of its class is created.

When a data member is declared as static, it can be accessed with the help of any one of the object of that class. In other words, a static data member is commonly shared all the objects of the same class.

note

The type and scope of each static member variable must be defined outside the class definition. This is necessary, because static data members are stored separately rather than as a part of an object. 

#include<stdio.h>
#inclide<conio.h>
#inclide<iomanip.h>
using namespace std;
class num
{
  private:
    static int n;
  public:
    void accept();
    void disp();
};
int num::n;
void num::accept()
{
  cin>>n;
}
void num::disp()
{
  cout<<n<<endl;
}
void main()
{
  num nb1,nb2;
  cout<<"Initial value in the First Object :";
  nb1.disp();
  cout<<"Initial value in the Second Object :";
  nb2.disp();
  cout<<"Enter any value into First Object :";
  nb1.accept();
  cout<<"Value in the First Object :";
  nb1.disp();
  cout<<"Value in the Second Object :";
  nb2.disp();
  cout<<"Enter any value into Second Object :";
  nb2.accept();
  cout<<"Value in the First Object :";
  nb1.disp();
  cout<<"Value in the second Object :";
  nb2.disp();
}
