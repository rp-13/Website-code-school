static member function

When a member function is declared of type static, then it is possible to invoke the member functions directly, with the help of class name without declaring any object of that class. A static function can have to access only other static members declare in the same class. A static member function can be called using classname as follows.

syntax

class_name::function_name([arguments_list]);

code

#include<stdio.h>
#inclide<conio.h>
class num
{
  private:
  int n;
  public:
  static void accept();
  static void disp();
  void display();
};
int num::n;
void num::accept()
{
  cin>>n;
}
void num::disp()
{
  cout<<n<<endl;
}
void num::display()
{
  cout<<n<<endl;
}
void main()
{
  num nb;
  cout<<"Initial data in the Object :";
  nb.display;
  cout<<"Enter any Number into static data member :";
  num::accept();
  cout<<"Given Number  in the static data member  :";
  num::disp();
  cout<<"value in the Object   :";
  nb.display();
}
